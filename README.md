# agds_report_ETadiri
This repository contains my reports (Digital Soil Mapping and Spatial Upscaling) for the University of Bern course Applied Geodata Science II.



Folder Structure:

data-raw
This folder contains raw data provided by geco-bern.

data
This folder contains processed data ready for analysis.

vignettes
This folder contains R markdown files which detail my report analyses and responses, and any knitted html documents from these reports.
